# Android Application | Siemsi

Android-based Siemsi fortune-telling application that uses motion sensors to detect shaking gestures and generate fortune results. The project was fully developed using Android Studio, including sensor logic, animations, sound effects, and UI/UX design.

## Project Overview

This project was developed as part of the **Mobile Application Development** course, a core subject in the third year of study.

The objective of the project was to design and develop a mobile application based on a topic of interest while applying concepts learned in Android development.

The application simulates a traditional **Siemsi fortune-telling experience**. Users shake their device as if shaking a real Siemsi container. The application detects the shaking motion using the device's motion sensors and then generates a fortune result accompanied by animations, sound effects, and vibration feedback.

The entire application was developed using **Android Studio**.

## Features

- Motion sensor detection for shaking gestures  
- Random fortune generation system  
- Animation effects during the shaking process  
- Sound effects and vibration feedback for interaction  
- Simple and user-friendly UI design

## My Role

- Designed the overall UI/UX of the application  
- Implemented motion sensor logic to detect shaking gestures  
- Developed the fortune generation mechanism  
- Added animation effects, sound effects, and vibration feedback  
- Implemented and tested the application using Android Studio
